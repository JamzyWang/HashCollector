# HashCollector

---
This repository collects general hash functions implemented by multiple languages such C,C++,Java,Python,Ruby,Pascal. Till now these hash functions include:

 - MD5
 - MurmurHash
 - Bloom filter
 - DJBHash
 - APHash
 - BKDRHash
 - BPHash
 - DEKHash
 - ELFHash
 - FNVHash
 - JSHash
 - PJWHash
 - RSHash
 - SDBMHash
